Recipe App

A full stack application created to help users learn new cooking recipes.

To use:

1. Download repository
2. Open up a terminal and run "npm run dev" while in the repository directory
3. Create a .env file and add "GEMINI_API_KEY = [Your GEMINI API key here]"
4. Open up another terminal and run "node server.js" to run the backend server

Note: Requires an API key from Google Gemini for the app to run succesfully.
